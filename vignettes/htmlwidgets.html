Demo list

To install (most of) the necessary packages to run the vignettes below,

  • Interactive Heat Maps: d3heatmap, R, Rmd
  • Interactive tables: DT, R, Rmd
  • Circle Plots with Bundled Edges: edgebundleR, R, Rmd
  • Formattable Data Structures: formattable, R, Rmd
  • Interactive ggplot2 graphs: ggiraph, R, Rmd
  • Network graphs: networkD3, R, Rmd
  • Add pan and zoom to graphics: svgPanZoom, R, Rmd
  • jQuery Sparkline HTML Widget:sparkline, R, Rmd

Other widgets

  • leaflet creates interactive map - homepage
  • pairsD3: d3 Scatterplot Matrices - demo
  • rhandsontable: Interface to the Handsontable.js Library; a minimalist Excel-like data grid editor - examples
  • scatterD3: d3 JavaScript Scatterplot from R - examples
  • threejs: Interactive 3D Scatter Plots, Networks and Globes - examples
  • visNetwork: Network Visualization using vis.js Library - examples

Entire plotting frameworks

The packages below link to very javascript frameworks (plotly is quite nice).

  • plotly: Create Interactive Web Graphics via plotly.js - examples
  • rAmCharts: JavaScript Charts API Tool - examples, about
  • rbokeh: R Interface for Bokeh
  • highcharter: A wrapper for the Highcharts library - homepage
  • Interface to the dygraphs JavaScript charting library - demos
  • metricsgraphics provides an htmlwidgets interface to the MetricsGraphics.js - examples

Niche widgets

Interesting widgets, but aimed at a particular market.

  • DiagrammeR: Create graph diagrams and flowcharts using R
  • qtlcharts: Interactive Graphics for QTL Experiments - examples
  • radarchart: Radar Chart from Chart.js
  • rchess: R package for chess validations!
  • treescape: Statistical Exploration of Landscapes of Phylogenetic Trees
